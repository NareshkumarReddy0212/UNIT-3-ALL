Experiment (e): Fetch and Render Data from a Public API using Axios Axios is a popular JavaScript library used to make HTTP requests from browsers and Node.js. In React, Axios helps fetch data from APIs and update components dynamically. In this experiment, students will use Axios to fetch posts from a public API (JSONPlaceholder) and display them in a React application. Aim: To fetch data from an external API and render it dynamically using React and Axios. Tools Required: • Node.js (LTS version) – https://nodejs.org/ • Visual Studio Code – https://code.visualstudio.com/ • Web browser (Google Chrome recommended) Procedure: 1. Install Axios in your React project using the command: npm install axios 2. Create a new component file named src/components/PostsList.js 3. Import useState and useEffect from React to manage state and side effects. 4. Use the useEffect hook to make a GET request using Axios to fetch posts from JSONPlaceholder API. 5. Store the response data in the posts state using the useState hook. 6. Render the list of post titles dynamically using the map() function. 7. Import and render the PostsList component inside App.js. 8. Run the project using npm start and verify that posts are displayed correctly.
